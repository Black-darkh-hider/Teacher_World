
Teacher World - Node.js Backend + Frontend (bundled)

How to run:
1. Open a terminal.
2. cd backend
3. npm install
4. npm start
5. Open browser to http://localhost:5000/contact.html (or open the HTML files from frontend folder)

Notes:
- OTPs are logged to the server console for demo.
- The app uses a SQLite database file at backend/database.db (auto-created).
